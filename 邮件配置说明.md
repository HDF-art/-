# 邮件服务器配置说明

## 配置文件位置
`backend/src/main/resources/application.yml`

## 配置示例 (QQ邮箱)

```yaml
spring:
  mail:
    host: smtp.qq.com
    port: 587
    username: your_email@qq.com
    password: your授权码
    properties:
      mail:
        smtp:
          auth: true
          starttls:
            enable: true
            required: true
    default-encoding: UTF-8
```

## 如何获取QQ邮箱授权码

1. 登录QQ邮箱
2. 设置 → 账户 → 开启IMAP/SMTP服务
3. 生成授权码

## 其他常用邮箱SMTP配置

### 163邮箱
- SMTP服务器: smtp.163.com
- 端口: 465 (SSL) 或 587 (TLS)

### Gmail
- SMTP服务器: smtp.gmail.com
- 端口: 587 (TLS)

### QQ企业邮箱
- SMTP服务器: smtp.exmail.qq.com
- 端口: 465 (SSL)
