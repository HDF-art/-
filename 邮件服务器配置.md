# 邮件服务器配置指南

## 当前状态
- 后端已集成邮件发送功能
- 需要配置有效的邮箱SMTP

## 支持的邮箱类型

### 1. QQ邮箱
```
SMTP服务器: smtp.qq.com
端口: 587 (TLS) 或 465 (SSL)
用户名: your_qq@qq.com
授权码: 在QQ邮箱设置中生成
```

### 2. 163邮箱
```
SMTP服务器: smtp.163.com
端口: 587 (TLS)
授权码: 在163邮箱设置中生成
```

### 3. Gmail
```
SMTP服务器: smtp.gmail.com
端口: 587 (TLS)
```

## 如何配置

1. 登录邮箱账号
2. 开启SMTP服务
3. 生成授权码
4. 修改配置文件: `backend/src/main/resources/application.yml`

## 测试发送

配置完成后，访问平台点击"获取验证码"测试。
